Read FISS data

This sub-package is used to read all FISS data and plot the raster and spectral profile. Using this module, you can draw the quick-look interactive images.

FISSpy.read package

Classes

ClassDescription
raw Read a raw object frame.
calibData Read a biasdark frame or flat frame.
FISS Read a processed (*1.fts) or compressed (*_c.fts) file.
AlignCube Read an alignCube file.

FISSpy.image package

Classes

ClassDescription
