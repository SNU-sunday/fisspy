FISSpy Documentation



Welcome to FISSpy's documentation!

FISSpy is the Python Package for analyzing the GST/FISS data. This package is released on a conda-forge channel of the Anaconda package repository and PyPi server. You can see the source code of the FISSpy on the GitHub. Among these servers, we recommend the conda-forge channel. Please see the details of the installation here.

The FISSpy package is the interactive analysis tool for the cube of FISS data. The code reference and how to handle the interactive figure are described below:
